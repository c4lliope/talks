A few of my favorite things...

  • Declarative, not Imperative.
    No more $(".list").append(...);
  • All the power of javascript in your views.
    you don't need to rely on clunky templating helpers like {{#each}} or {{>}}.
  • Re-render the entire app on every change.
    We don't need to declare dependencies for our components' properties — React treats all of our code as a black box.
  • Speed, you say? Virtual DOM, I reply!
    React generates a virtual DOM tree on updates, diffs it with the existing tree, and performs a minimal set of changes.
    This is often faster than manual DOM manipulation. (See the announcement video above)
